# W25ANN

`W25ANN` is the python mlflux package (machine learning for air-sea flux parameterization) from [Wu et al. 2025](https://doi.org/10.48550/arXiv.2503.03990) study.

- The code here is a copy of the original repository [mlflux](https://github.com/jiarong-wu/mlflux).

- Main contact to this work is:
    - Jiarong Wu: jiarong.wu@nyu.edu

***

Install by running

`pip install -e .`
